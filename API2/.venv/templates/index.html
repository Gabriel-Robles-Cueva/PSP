FUNCIONAMIENTO DE LA API

Esta es una API sobre los objetos de pokémon pensada para el uso de estudio para combates Pokémon aqui se pueden ver los objetos del juego, puedes filtrar según lo que mas te interese buscar y puedes mirar su precio de mercado en POKEMMO.

Cada boton lleva un formulario para llevar a cabo su función.

  1. AÑADIR OBJETO
    Página en la que hay un formulario para introducir un objeto con sus datos y debajo una tabla donde se puede comprobar que se ha añadido correctamente y tambien puedes eliminar los objetos.
  2. VER LISTA DE OBJETOS
    Aquí puedes ver la lista entera de objetos en modo JSON.
    http://127.0.0.1:5000/get-objects
  3. VER LISTA DE OBJETOS FILTRADO POR NOMBRE
    Aquí puedes ver la lista entera de objetos por nombre en modo JSON.
    http://127.0.0.1:5000/get-objects-by-name?nombre=Pocion
  4. VER LISTA DE OBJETOS FILTRADO POR TIPO
    Aquí puedes ver la lista entera de objetos por tipo en modo JSON.
    http://127.0.0.1:5000/get-objects-by-tipo?tipo=Curacion
  5. ELIMINAR OBJETOS POR NOMBRE
    Aquí puedes borrar un objeto por nombre, debes cambiar "EJEMPLO" por el nombre del objeto.

    http://127.0.0.1:5000/delete-object-by-name?nombre=EJEMPLO

  6. EDITAR TIPO DE UN OBJETO
    Aquí puedes editar el tipo de un objeto.
    http://127.0.0.1:5000/edit-object-type?id=9&tipo=Bayas
  7. EDITAR PRECIO DE UN OBJETO
    Aquí puedes editar el precio de un objeto.
    http://127.0.0.1:5000/edit-object-price?id=7&precio=200
