FUNCIONAMIENTO DE LA API

Esta es una API sobre los objetos de pokémon pensada para el uso de estudio para combates Pokémon aqui se pueden ver los objetos del juego, puedes filtrar según lo que mas te interese buscar y puedes mirar su precio de mercado en POKEMMO.

Cada boton lleva un formulario para llevar a cabo su función.

  1. AÑADIR OBJETO
    Página en la que hay un formulario para introducir un objeto con sus datos y debajo una tabla donde se puede comprobar que se ha añadido correctamente y tambien puedes eliminar los objetos.
  2. VER LISTA DE OBJETOS
    Aquí puedes ver la lista entera de objetos en modo JSON.
